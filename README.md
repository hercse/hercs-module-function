# hercs-module-function
自己用的模組化功能

## 待製作的功能
- 整合所有``/js``裡面的資料。
- 選單開關端口。(開關訊號發送、已開啟時間。)

### style.scss

```
<link rel="stylesheet" href="https://module-function.herc.se/scss/style.css">
```

### cursor.js

```
cursor((e) => {
            // function here, when mouse moveing.
            console.log(e);
        }, "body"); // defult give css variable to "body".
```
